# UNIArt [Unity 编辑器拓展]

## 功能概览
- 美术工作目录规范化
- UNIHper.Ghost 组件编辑器拓展
- Animator 动画管理

- alt+1 打开工作台
- alt+2 查看页面列表
- alt+3 定位当前正在编辑的预制体
- alt+拖拽项目资源 在UNIArt内保持依赖拖拽
- ctrl 拖拽Asset资源，保持预制体派生
- alt 子资源拖拽
  

## 原则
 - 模板视图的资源，作为公用组件使用，尽可能减少被修改的可能
